Тестовый бот для https://t.me/lztrox (1868383153)


ТЗ:
/start - "правила" с инлнайн кнопкой согласиться (галочка)

После соглашения появляется 3 кнопки:
первый ряд: мои строки
второй ряд: добавить строку, проверить баланс

"мои строки" - вывод всех строк в одном сообщении

"добавить" - запрашивается ввод сид фразы (Проверка на формат bip39, другие идут в ошибку)
Можно загружать пачкой (Пример - 
строка
строка
строка
)

При удачном добавлении сообщение добавляется в базу и отправляется админу в формате {
айди юзера
кол-во загруженных строк
}

"проверить баланс" - проверка баланса eth адреса кошелька
